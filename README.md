# 🛍️ Fake Store

A modern and responsive **Fake Store** web application built using **React.js**. The project displays products in an attractive interface and demonstrates fundamental concepts of React such as components, routing, state management, and reusable UI design.

## 🚀 Features

- 🏠 Home page with a clean user interface
- 🛒 Product listing page
- 🔗 Navigation using React Router
- 📱 Responsive design
- ⚡ Fast development with Vite
- ♻️ Reusable React components
- 🎨 Simple and user-friendly layout

## 🛠️ Tech Stack

- React.js
- React Router DOM
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/keertig15/Fakestore.git
```

### Navigate to the project directory

```bash
cd Fakestore
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
